Tidas Landing Page
==================================================

Getting Started
--------------------------------------------------
Bower install:
```
bower install
```
NPM install:
```
sudo npm install
```

Start Gulp:
```
gulp
```

Visit page:
```
http://localhost:3000
http://192.168.1.104:3000
```

Visit UI:
```
http://localhost:3001
http://192.168.1.104:3001
```
